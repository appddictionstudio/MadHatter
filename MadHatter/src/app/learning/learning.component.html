Advanced Software Developer

This is a 15 week bootcamp that takes experienced developers and teaches them the tools to work in a agile full stack team. Some technologies covered will be: Pivotal Cloud Foundry, Java, Angular, J Unit, and Fortify. The course will revolve around launching a mock enterprise level application. E.g. a store management or base management system. Students will model the agile work flow by first setting up their application on the cloud so that it is publicly accessible, and then using CI/CD principles to continuously release new features of the application over the course of the bootcamp. Throughout this, they will be writing their code in accordance with Test Driven Development and standard security principles.

UI / UX Bootcamp

In this 12 week Bootcamp, learners will gain a solid understanding of the fundamentals of UI/UX design. Learners will discover user-centric design, prototyping, engaging stakeholders, and how to solve real world design problems. They will learn how to properly research a user experience with the three principles of Inspiration, Ideation, and Implementation. Learners will discover how to implement Lean UX for Agile working environments
